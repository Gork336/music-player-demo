配置数据库信息

- localhost 是你要连接的地址
- test 为你的库名
- username 用户名
- password 密码

```properties
spring.datasource.url=jdbc:mysql://localhost/test
spring.datasource.username=dbuser
spring.datasource.password=dbpass
```

`spring.jpa.hibernate.ddl-auto=update` 可以根据代码自动更新表，不用手动创建表
